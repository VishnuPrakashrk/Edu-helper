## Overview
EduHelperAgent is a simple educational chatbot built using Laravel 12.
It helps school students learn basic topics in a safe and controlled way.

Supported topics:
- Solar System
- Fractions
- Water Cycle

This project follows a LarAgent-style custom agent architecture.

---

## Tech Stack
- Laravel 12
- PHP 8+
- REST API
- Session-based conversation memory
